# Semantics of Funnel plots

A funnel plot is expected to have many attributes, some MUST (mandatory), some SHOULD (desirable) and some MAY (optional).

## Structure of a typical plot:

| a | b | c |
| ------------- | -- | -------- |
| x | Header | x |
| x | Plot_Top | x |
| Plot_Left  | Plot  | Plot_Right |
| x | Plot_Bottom  | x |
| x | Footer | x |
